# Urbana High School Class of 2006 Reunion Website

A Vercel-ready Next.js preview site.

## Easiest way to deploy

1. Unzip this folder.
2. Create a new GitHub repository and upload the folder contents.
3. In Vercel, click **Add New → Project**.
4. Import the GitHub repository.
5. Vercel will detect Next.js automatically. Click **Deploy**.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.
